    # **出血术** *Bleed*

    **学派** 死灵系 

    **环位** 牧师/先知 0, 审判者 0, 通灵者 0, 催眠师 0, 秘学士 0, 异能者 0, 萨满 0, 术士/法师 0, 唤魂师 0, 女巫 0

    ## 🪄 施放

    **施法时间** 1 标准动作

    **成分** 语言, 姿势

    ## ✨ 效果 

    **目标** 一个活物 

    **距离** 近距 (25尺+5尺/2等级)  

    **持续时间** 立即 

    **豁免** 意志, 通过则无效

    **法术抗力** 是

    ## 📖 描述

    你致使一名HP低于0点但是已经稳定的生物重新面临死亡。当施放该法术时，你要选择一名HP为-1或更低的活物作为目标。该名生物会开始死亡，每回合受到1点伤害。这名生物能在之后如常稳定伤势。该法术会使濒死的生物受到1点伤害。
