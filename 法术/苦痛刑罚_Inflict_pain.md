    # **苦痛刑罚** *Inflict pain*

    **学派** 惑控系 \[影响心灵，痛苦\] 

    **环位** 反圣武士 2, 审判者 2, 通灵者 2, 催眠师 2, 秘学士 2, 异能者 2, 术士/法师 2, 唤魂师 2, 女巫 3

    ## 🪄 施放

    **施法时间** 1 标准动作

    **成分** 语言, 姿势

    ## ✨ 效果 

    **目标** 一个生物 

    **范围** 近距（25尺 + 每2个等级5尺）

    **距离** 近距 (25尺+5尺/2等级)  

    **持续时间** 1轮/等级 

    **豁免** 意志，通过则部分生效；见后文

    **法术抗力** 是

    ## 📖 描述

    通过心灵感应，你使用极端的苦痛来伤害目标的精神与肉体，这使得它在攻击检定、技能检定和属性检定中承受-4减值。成功通过意志豁免能够将该法术的持续时间降低至1轮。
