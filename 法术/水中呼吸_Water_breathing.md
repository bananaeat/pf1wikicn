# **水中呼吸** *Water breathing*

**学派** 变化系 

**领域** 水领域 3

**环位** 炼金术师 3, 血脉狂怒者 3, 牧师/先知 3, 德鲁伊 3, 猎人 3, 魔战士 3, 异能者 3, 萨满 3, 术士/法师 3, 召唤师 3, unchained 召唤师 3

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势, 材料, 圣徽

## ✨ 效果 

**目标** 接触到的活物 

**距离** 接触  

**持续时间** 2小时/等级; 见后文 

**豁免** 意志, 通过则无效 （无害）

**法术抗力** 是

## 📖 描述

受术者可在水下自由呼吸。总持续时间需要平均分给所有你接触到的生物。此法术不会让受术者无法在空气中呼吸。
