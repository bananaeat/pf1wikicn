# **治疗重伤** *Cure serious wounds*

**学派** 咒法系 (医疗) 

**环位** 导师 3, 炼金术师 3, 吟游诗人 3, 牧师/先知 3, 德鲁伊 4, 猎人 4, 审判者 3, 秘学士 3, 圣骑士 4, 游侠 4, 萨满 3, 唤魂师 3, 女巫 4

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势

## ✨ 效果 

**目标** 接触到的生物 

**距离** 接触  

**持续时间** 立即 

**豁免** 意志, 通过则减半 （无害）

**法术抗力** 是

## 📖 描述

该法术的功能如同‘治疗轻伤 （cure light wounds）’一般，不过它能治疗 {3d8+CL （最高+15）} 点伤害。
