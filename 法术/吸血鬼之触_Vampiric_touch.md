# **吸血鬼之触** *Vampiric touch*

**学派** 死灵系 

**领域** 鲜血子域 3, 邪魔子域 3

**环位** 反圣武士 3, 血脉狂怒者 3, 魔战士 3, 通灵者 3, 催眠师 3, 异能者 3, 术士/法师 3, 唤魂师 3, 女巫 3

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势

## ✨ 效果 

**目标** 接触到的活物 

**距离** 接触  

**持续时间** 立即/1 小时; 见后文 

**豁免** 无

**法术抗力** 是

## 📖 描述

你必须成功进行一次近战接触攻击。你的接触会造成每两个CL 1d6点伤害 （最高10d6）。你获得等同于造成的伤害点临时HP。你获得的HP不能超过 {受术者的当前HP+受术者的体质属性} （这已经足矣杀死受术者）。这些临时HP会在1小时后消散。
