    # **共用回力武器** *Returning weapon, communal*

    **学派** 咒法系 (传送) 

    **环位** 吟游诗人 3, 牧师/先知 3, 审判者 2, 魔战士 2, 游侠 2, 术士/法师 3, 女巫 3

    ## 🪄 施放

    **施法时间** 1 标准动作

    **成分** 语言, 姿势

    ## ✨ 效果 

    **目标** 接触到的复数武器 

    **距离** 近距 (25尺+5尺/2等级)  

    **持续时间** 1 minute/level 

    **豁免** Will negates (harmless, object)

    **法术抗力** 是

    ## 📖 描述

    这个法术工作方式如同回力武器 （Returning Weapon），但你可以把持续时间以1分钟为单位划分给多把武器。
