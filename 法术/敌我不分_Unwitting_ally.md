# **敌我不分** *Unwitting ally*

**学派** 惑控系 (魅惑) \[影响心灵\] 

**环位** 吟游诗人 0

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势

## ✨ 效果 

**目标** 一名活物 

**距离** 近距 (25尺+5尺/2等级)  

**持续时间** 1轮 

**豁免** 意志，通过则无效

**法术抗力** 是

## 📖 描述

通过迷惑目标的心智。目标会在短时间内难以分辨敌友。在判断夹击时，受术者被视为你的盟友，而非敌人的盟友。受术者不会由于该法术的效果进行其他敌对行动对抗你的敌人。
