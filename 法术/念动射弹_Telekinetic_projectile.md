# **念动射弹** *Telekinetic projectile*

**学派** 塑能系 

**环位** 秘学士 0, 异能者 0, 唤魂师 0

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势

## ✨ 效果  

**距离** 近距 (25尺+5尺/2等级)  

**持续时间** 立即 

**豁免** 无

**法术抗力** 否

## 📖 描述

你将一个至多5磅重的物体掷向目标。你必须成功通过远程攻击（而非远程接触攻击）来命中目标；若命中，你会对物体和目标同时造成1d6点钝击伤害。被投掷的物体类型不会改变伤害类型或攻击的其他特性，即便你以此方式抛掷武器或魔法物品也是如此。
