    # **精怪之相** *Haunted fey aspect*

    **学派** 幻术系 (五官幻觉) 

    **环位** 吟游诗人 0, 术士/法师 0

    ## 🪄 施放

    **施法时间** 1 标准动作

    **成分** 姿势

    ## ✨ 效果  

    **范围** 个人

    **距离**   

    **持续时间** 1轮/等级 （可解消） 

    **法术抗力** 否

    ## 📖 描述

    你用一层奇怪的幻觉覆盖自己，使你看起来像一个扭曲怪异的精类生物。

    你在针对一次单次攻击，获得DR1/寒铁的伤害减免，持续到法术持续时间结束或受到一次伤害。
