    # **流逝荣耀** *Bleed glory*

    **学派** 死灵系 \[诅咒\] 

    **环位** 牧师/先知 3, 术士/法师 3, 女巫 3

    ## 🪄 施放

    **施法时间** 1 标准动作

    **成分** 语言, 姿势

    ## ✨ 效果 

    **目标** 1个神话生物 

    **距离** 近距 (25尺+5尺/2等级)  

    **持续时间** 1分钟/级 

    **豁免** 意志，通过则无效

    **法术抗力** 是

    ## 📖 描述

    目标的所有道途能力、专长、法术以及其他效果所花费的神话之力增加1点。
