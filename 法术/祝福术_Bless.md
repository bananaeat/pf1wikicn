    # **祝福术** *Bless*

    **学派** 惑控系 (胁迫) \[影响心灵\] 

    **领域** 团队领域 1, 领导子域 1, 决意子域 1

    **环位** 导师 1, 牧师/先知 1, 审判者 1, 圣骑士 1, 萨满 1

    ## 🪄 施放

    **施法时间** 1 标准动作

    **成分** 语言, 姿势, 圣徽

    ## ✨ 效果  

    **范围** 施法者，以及以施法者为中心半径50尺爆发区域内的所有盟友

    **距离** 50尺  

    **持续时间** 1分钟/等级 

    **豁免** 无

    **法术抗力** 是

    ## 📖 描述

    祝福术让你的盟友充满了勇气。每个盟友都在攻击检定和对抗恐惧的豁免检定中获得+1士气加值。祝福术反制和解除‘绝望术 （bane）’
