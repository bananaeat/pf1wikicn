    # **打击死灵** *Disrupt undead*

    **学派** 死灵系 

    **环位** 审判者 0, 魔战士 0, 术士/法师 0

    ## 🪄 施放

    **施法时间** 1 标准动作

    **成分** 语言, 姿势

    ## ✨ 效果  

    **距离** 近距 (25尺+5尺/2等级) 

    **效果** 射线 

    **持续时间** 立即 

    **豁免** 无

    **法术抗力** 是

    ## 📖 描述

    你射出一股正能量射线。你必须通过远程接触攻击来命中目标，若射线命中不死生物，则会造成1d6点伤害。
