# **犯罪浪潮** *Crime wave*

**学派** 惑控系 (胁迫) \[影响心灵\] 

**环位** 反圣武士 4, 吟游诗人 5, 催眠师 5, 异能者 7, 术士/法师 7

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势, 材料

## ✨ 效果 

**目标** 1个生物/级，任意两者之间的距离不超过30尺 

**范围** 中距（100尺 + 10尺/级）

**距离** 中距 (100尺+10尺/等级)  

**持续时间** 1轮/等级（可解消） 

**豁免** 意志，通过则无效

**法术抗力** 是

## 📖 描述

你向目标灌输了无法抑制的贪欲，促使他们发起疯狂地犯罪行为。受到犯罪浪潮影响的生物必须每轮投掷百分骰，并以此决定他们会采取的行动。
