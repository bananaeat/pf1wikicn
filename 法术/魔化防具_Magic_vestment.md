# **魔化防具** *Magic vestment*

**学派** 变化系 

**领域** 高贵领域 3, 力量领域 3, 战争领域 3

**环位** 牧师/先知 3, 审判者 3, 秘学士 3, 萨满 3

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势, 圣徽

## ✨ 效果 

**目标** 接触到的护甲或盾牌 

**距离** 接触  

**持续时间** 1小时/等级 

**豁免** 意志, 通过则无效 （无害, 物品）

**法术抗力** 是

## 📖 描述

你可以对一套护甲或盾牌赋予每四个CL+1增强加值 （最高为20级时的+5）。

在判断该法术的功能时，普通服装被视为没有AC加值的护甲。
