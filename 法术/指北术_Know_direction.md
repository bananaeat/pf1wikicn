    # **指北术** *Know direction*

    **学派** 预言系 

    **环位** 吟游诗人 0, 德鲁伊 0, 猎人 0, 秘学士 0, 异能者 0, 萨满 0

    ## 🪄 施放

    **施法时间** 1 标准动作

    **成分** 语言, 姿势

    ## ✨ 效果 

    **目标** 自身 

    **距离** 自身  

    **持续时间** 立即 

    **法术抗力** 否

    ## 📖 描述

    当你施放该法术时，会立刻得知从你所在的位置起哪个方向是北方。该法术在任何存在“北方”概念的环境中都有效，但是在异界中可能失去效果。该法术只是在你施展的瞬间使你知道北方的位置，如果你没有借助参照物来保持确定方向的话，你有可能再次迷失方向。
