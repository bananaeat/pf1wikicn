# **净化食粮** *Purify food and drink*

**学派** 变化系 

**环位** 导师 0, 牧师/先知 0, 德鲁伊 0, 猎人 0, 秘学士 0, 萨满 0

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势

## ✨ 效果 

**目标** 每等级1立方尺被污染的食物和水 

**距离** 10尺  

**持续时间** 立即 

**豁免** 意志, 通过则无效 （物品）

**法术抗力** 是

## 📖 描述

这个法术能把变质、腐臭、染疫、有毒或遭受其他污染的食物和水净化为可食用和饮用状态。这个法术不能阻止之后的自然腐败或变质。邪水和类似意义的食物和饮料被净化食粮处理后会失去作用，但是这个法术对任何类型生物或魔法药剂都无效。每加仑水重约8磅。一立方尺水大约为8加仑，重60磅。
