# **燃烧之手** *Burning hands*

**学派** 塑能系 \[火\] 

**领域** 火领域 1

**环位** 导师 1, 血脉狂怒者 1, 魔战士 1, 秘学士 1, 萨满 1, 术士/法师 1, 女巫 1

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势

## ✨ 效果  

**范围** 锥形爆发区域

**距离** 15尺  

**持续时间** 立即 

**豁免** 反射, 通过则减半

**法术抗力** 是

## 📖 描述

锥形的灼热火焰从你的指尖喷出。所有在火焰区域内的生物都要受到每CL 1d4点火焰伤害 （最多5d4）。如果火焰接触到可燃材料，它们会烧起来。角色能以一个整轮动作扑灭燃烧的物品。
