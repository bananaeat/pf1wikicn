# **强迫凝视** *Lock gaze*

**学派** 惑控系 (胁迫) \[影响心灵\] 

**环位** 吟游诗人 1, 审判者 1, 魔战士 1, 术士/法师 1, 女巫 1

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势

## ✨ 效果 

**目标** 一名生物 

**距离** 近距 (25尺+5尺/2等级)  

**持续时间** 1轮/等级 

**豁免** 意志，通过则无效

**法术抗力** 是

## 📖 描述

你强迫目标看着并只能看着你的眼睛直到法术效果消失或被解除。

在盯着你时，目标将会无视其他所有生物而只注视你，从而使其他生物对它攻击时视作隐蔽。

如果目标主动将视线移离你的眼睛，它将会目盲一轮并且法术结束。

如果你主动将视线移离目标生物的眼睛，或是陷入昏迷或死亡，目标则不会受到任何影响。

没有视力和免疫凝视攻击的生物对该法术免疫。
