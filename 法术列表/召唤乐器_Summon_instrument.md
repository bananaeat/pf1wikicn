# **召唤乐器** *Summon instrument*

**学派** 咒法系 (召唤) 

**环位** 吟游诗人 0

## 🪄 施放

**施法时间** 1 轮

**成分** 语言, 姿势

## ✨ 效果  

**距离** 0尺 

**效果** 一个被召唤的手持乐器 

**持续时间** 1分钟/等级 

**豁免** 无

**法术抗力** 否

## 📖 描述

该法术会按照你的选择召唤一件手持乐器。乐器将在你手中或脚边出现 （由你选择）。它与普通乐器相同。每次施法只能召唤一件乐器，且仅供施法者演奏。你不能召唤出大到无法由两手持用的乐器。被召唤的乐器会在法术结束后消失。
