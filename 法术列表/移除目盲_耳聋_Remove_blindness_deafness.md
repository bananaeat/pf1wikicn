# **移除目盲/耳聋** *Remove blindness/deafness*

**学派** 咒法系 (医疗) 

**领域** 纯净子域 3

**环位** 炼金术师 3, 牧师/先知 3, 催眠师 3, 圣骑士 3, 萨满 3, 唤魂师 3, 女巫 3

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势

## ✨ 效果 

**目标** 接触到的生物 

**距离** 接触  

**持续时间** 立即 

**豁免** 强韧, 通过则无效 （无害）

**法术抗力** 是

## 📖 描述

移除目盲/耳聋能够治愈目盲或者耳聋 （依照你的选择），不论这目盲和耳聋的效果是魔法还是自然造成的。本法术不会恢复耳朵或眼睛失去的部分，但是能修复受损伤的部分。

移除目盲/耳聋反制和解除‘目盲/耳聋术 （blindness/deafness）’。
