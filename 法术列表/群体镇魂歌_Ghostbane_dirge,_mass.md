# **群体镇魂歌** *Ghostbane dirge, mass*

**学派** 变化系 

**环位** 吟游诗人 4, 牧师/先知 5, 审判者 5, 圣骑士 3

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势, 材料, 圣徽

## ✨ 效果 

**目标** 1个虚体生物/等级，互相之间不得超过30尺 

**距离** 近距 (25尺+5尺/2等级)  

**持续时间** 1 round/level 

**豁免** Will negates

**法术抗力** 是

## 📖 描述

这个法术工作方式如同镇魂歌 （Ghostbane Dirge），不过能够影响复数目标。
