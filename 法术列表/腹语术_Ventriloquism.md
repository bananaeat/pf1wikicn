# **腹语术** *Ventriloquism*

**学派** 幻术系 (虚假幻觉) 

**环位** 吟游诗人 1, 通灵者 1, 催眠师 1, 异能者 1, 术士/法师 1, 唤魂师 1, 召唤师 1, unchained 召唤师 1

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 法器 (卷成喇叭的羊皮纸)

## ✨ 效果  

**距离** 近距 (25尺+5尺/2等级) 

**效果** 有意义的声音，通常是语言 

**持续时间** 1分钟/等级 

**豁免** 意志,通过则不相信 （若有互动）

**法术抗力** 否

## 📖 描述

可以使你的话语 （或其它你可以用嘴巴发出的声音） 听起来像是从别的地方发出的。你可以使用任何你会说的语言。关于这些声音，若任何听到它们的生物通过豁免检定，则会认为听到的是幻觉 （但仍听得到声音）。
