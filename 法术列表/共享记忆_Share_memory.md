# **共享记忆** *Share memory*

**学派** 预言系 

**环位** 吟游诗人 2, 术士/法师 2, 女巫 2

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势

## ✨ 效果 

**目标** 你和一名接触到的生物 

**距离** 接触  

**持续时间** 立即 

**豁免** 意志，通过则无效

**法术抗力** 是

## 📖 描述

你暂时将你的思想与目标连接到一起并且可以共享一段不超过一分钟长度的记忆。

你可以向目标展示你的一段记忆，向目标展示它自己的一段记忆，或者查看一段目标的记忆。
