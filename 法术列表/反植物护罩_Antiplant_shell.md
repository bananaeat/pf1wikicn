# **反植物护罩** *Antiplant shell*

**学派** 防护系 

**环位** 德鲁伊 4, 猎人 4

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势, 圣徽

## ✨ 效果  

**范围** 以你为中心, 10尺半径发散区域

**距离** 10尺  

**持续时间** 1分钟/等级 

**豁免** 无

**法术抗力** 是

## 📖 描述

反植物护罩产生了一个隐形的活动屏障，能保护护罩内的所有生物不受植物生物或活化植物的攻击。和大多数防护系法术一样，强行将此屏障逼向无法靠近该法术的生物会导致此屏障崩溃。
