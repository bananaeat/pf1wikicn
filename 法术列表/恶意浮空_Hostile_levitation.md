# **恶意浮空** *Hostile levitation*

**学派** 变化系 

**环位** 术士/法师 3, 女巫 3

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势, 材料

## ✨ 效果 

**目标** 一名生物 

**距离** 近距 (25尺+5尺/2等级)  

**持续时间** 1轮/等级 

**豁免** 意志，通过则无效

**法术抗力** 是

## 📖 描述

你让目标稍稍浮空，使其不能脚踏实地并移动速度减半。

目标对抗冲撞、拖曳和复位的CMD获得-4罚值。

目标生物使用近战或远程攻击会让自身失去平衡，它的第一次攻击检定有-1罚值，第二次-2，最高-5。它可以花费一个整轮动作来稳定平衡，这会让罚值重新从-1开始计算。

本法术对飞行中的生物无效。
