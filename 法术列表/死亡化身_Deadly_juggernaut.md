# **死亡化身** *Deadly juggernaut*

**学派** 死灵系 \[死亡\] 

**环位** 反圣武士 3, 牧师/先知 3, 审判者 3, 圣骑士 3

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势

## ✨ 效果 

**目标** 自己 

**范围** 个人

**距离** 自身  

**持续时间** 1分钟/等级 

**法术抗力** 否

## 📖 描述

随着你的敌人一个个倒下，你变得愈发危险和势不可挡。

在法术持续期间，每当你用近战攻击将一个够格的对手的生命值减少到0或以下，你的近战投骰、近战伤害投骰、力量检定和基于力量的技能检定便会获得+1幸运加值，并且获得DR2/- （最大+5和DR10/-）

一个合格的对手要求HD大于等于你的HD-4。
