# **移位术** *Displacement*

**学派** 幻术系 (五官幻觉) 

**领域** 星灵子域 3

**环位** 炼金术师 3, 吟游诗人 3, 魔战士 3, 通灵者 3, 催眠师 3, 秘学士 3, 异能者 3, 术士/法师 3, 唤魂师 3, 召唤师 3, unchained 召唤师 3

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 材料

## ✨ 效果 

**目标** 接触到的生物 

**距离** 接触  

**持续时间** 1轮/等级 

**豁免** 意志, 通过则无效 （无害）

**法术抗力** 是

## 📖 描述

该法术的受术者出现在距离其实际位置的大约2尺的地点。这使得他如同具有全隐蔽一般，获得50%失手率的优势。但和全隐蔽不同，移位术无法防止敌人如常以受术者为目标发动攻击。‘真知术 （true seeing）’可揭露受术者的真正位置并使失手率无效。
