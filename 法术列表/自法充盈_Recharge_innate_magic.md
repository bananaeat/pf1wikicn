# **自法充盈** *Recharge innate magic*

**学派** 变化系 

**环位** 炼金术师 1, 吟游诗人 1, 牧师/先知 1, 德鲁伊 1, 审判者 1, 魔战士 1, 术士/法师 1, 女巫 1

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势

## ✨ 效果 

**目标** 自己 

**距离** 自身  

**持续时间** 立即 

**法术抗力** 否

## 📖 描述

你将魔法能量引导至自身体内，填充你内在的魔法能力。所有你能够使用并且通过种族特性获得的0环以及1环类法术能力全部重新获得1次使用机会。
