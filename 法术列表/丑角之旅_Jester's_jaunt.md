# **丑角之旅** *Jester's jaunt*

**学派** 咒法系 (传送) 

**环位** 吟游诗人 3

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势

## ✨ 效果 

**目标** 一名活物 

**距离** 接触  

**持续时间** 立即 

**豁免** 意志，通过则无效

**法术抗力** 是

## 📖 描述

你将目标传送至你能看到、并且处于距目标30尺内的区域。目的地必须在坚实的地面之上，并且该次传送的终点空间本身对你传送的生物来说不能有害。
