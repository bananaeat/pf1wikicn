# **祈祷术** *Prayer*

**学派** 惑控系 (胁迫) \[影响心灵\] 

**领域** 团队领域 3, 亚空子域 3, 领导子域 3

**环位** 牧师/先知 3, 审判者 3, 圣骑士 3

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势, 圣徽

## ✨ 效果  

**范围** 以你为中心，40尺半径爆发范围内的所有盟友与敌人

**距离** 40尺  

**持续时间** 1轮/等级 

**豁免** 无

**法术抗力** 是

## 📖 描述

你将神之恩宠带给你自己和你的盟友，同时将厌恶带给你的敌人。你和你的每个盟友在攻击检定、武器伤害检定、豁免检定和技能检定上都获得+1幸运加值，同时你的每个敌人都会在这些检定上获得-1减值。
