# **造成中伤** *Inflict moderate wounds*

**学派** 死灵系 

**环位** 反圣武士 3, 牧师/先知 2, 审判者 2, 秘学士 2, 萨满 2, 唤魂师 2, 女巫 2

## 🪄 施放

**施法时间** 1 标准动作

**成分** 语言, 姿势

## ✨ 效果 

**目标** 接触到的生物 

**距离** 接触  

**持续时间** 立即 

**豁免** 意志, 通过则减半

**法术抗力** 是

## 📖 描述

该法术的功能如同‘造成轻伤 （inflict light wounds）’一般，不过你能造成 {2d8+CL （最高+10）} 点伤害。
