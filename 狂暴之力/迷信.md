# ![PFS](/pfs.webp =30x30) 迷信 (Superstition, Ex)

**益处：** 野蛮人对抗法术、超自然能力与类法术能力的豁免检定获得+2士气加值。此加值每4个野蛮人等级会再+1。当狂暴时，野蛮人无法自愿成为任何法术的目标，且必须进行豁免检定来对抗任何法术，就算它是由友方所施展的。
