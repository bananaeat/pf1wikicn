# ![PFS](/pfs.webp =30x30) 生生不息 (Renewed Life, Ex)

**先决条件** [野蛮人](/野蛮人)6级，[生机盎然](/狂暴之力/生机盎然)

**益处：** 在狂暴时，野蛮人可以忽略相当于他等级四分之一的临时负向等级。此外，每日一次，当野蛮人带着负向等级结束狂暴时，他可以尝试透过一个强韧检定来消除相当于他可以忽略等级数的负向等级。
