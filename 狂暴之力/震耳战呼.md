# ![PFS](/pfs.webp =30x30) 震耳战呼 (Battle Roar, Su)

**益处：** 每当角色成功的使用威吓凝视（Intimidating Glare）狂暴之力挫败了一个对手的士气，他就会同时对目标造成1d6点音波伤害。角色必须达到6级，并拥有威吓凝视狂暴之力才能选择本狂暴之力。
