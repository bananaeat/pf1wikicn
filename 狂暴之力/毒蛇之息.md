# ![PFS](/pfs.webp =30x30) 毒蛇之息 (Viper's Breath, Ex)

**先决条件** [毒蛇之吻](/狂暴之力/毒蛇之吻)

**益处：** 当嘴中含着一剂毒药时，野蛮人可以将这些毒药呼出在15尺锥形范围内。任何被毒素喷雾影响的生物必须通过DC为{毒药DC-4}的豁免检定，否则立刻受到毒素的影响。
