# 饕餮啮咬 (Feasting Bite, Ex)

**先决条件** 半兽人，[血腥啮咬](/狂暴之力/血腥啮咬)，[野性之怒](/狂暴之力/野性之怒) 或 一次天生啮咬攻击。

**益处：** 当野蛮人用啮咬攻击确认重击时，他恢复造成伤害一半的生命值。启动此能力会消耗一轮狂暴轮数。
