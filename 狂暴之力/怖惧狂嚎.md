# 怖惧狂嚎 (Terrifying Howl, Ex)

**益处：** 以一个标准动作，野蛮人能自口中爆出一声震人心魄的狂嚎声。30尺内的所有已战栗的敌人必须进行一次DC为“10+1/2野蛮人等级+力量修正”的意志豁免，失败者将恐慌“1d4+1”轮。当敌人对你的【怖惧狂嚎】进行过检定（无论成功通过与否），则它在接下来的24小时内都将免疫此效果。野蛮人必须先拥有【威吓凝视】且至少8级才能选取此狂暴之力。
